The monthy hall problem simulation

A simple simulation of the famous Monthy hall problem.

It is a famous probability puzzle where a contestant on a game show picks one of three doors. Behind one door is a car, and behind the other two are goats. 
The host, Monty Hall, reveals a goat behind one of the unchosen doors and offers the contestant the option to switch doors. 
You should always switch. Switching doubles your chances of winning from 1/3 to 2/3.

Run the program, and then see the printout for the statistical proof that switching is more likely to win.

Requirements

Python 3.x

How to Run

python main.py
